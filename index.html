Factor Risk Models

A factor model separates portfolio returns into shared exposures — market, industry, value, momentum, volatility — and idiosyncratic stock-specific returns.

Trading use: detect hidden bets, hedge unintended factor exposures, and explain whether PnL came from the desired edge.

exposures = estimate_factor_exposures(universe)
risk = covariance(factor_returns)
portfolio_risk = weights.T @ exposures @ risk @ exposures.T @ weights
if exposure(weights, "market") > limit:
    weights = hedge_factor(weights, factor="market")

Alpha Sizing

Alpha scores or expected returns are not final portfolio weights. They must be scaled by risk, confidence, liquidity, and diversification.

Trading use: convert a ranked signal into investable position sizes instead of blindly buying the top names.

alpha = model.predict_expected_returns(data)
confidence = estimate_signal_confidence(alpha)
vol = estimate_asset_volatility(returns)
raw_weight = alpha * confidence / vol
weights = normalize_to_gross(raw_weight, target_gross=1.0)

Factor-Neutral Portfolio Construction

Modify raw alpha-driven weights as little as possible while meeting constraints: factor neutrality, gross exposure, concentration, and turnover.

Trading use: keep the stock-selection edge while removing unwanted market or style bets.

raw = alpha_to_weights(alpha)
constraints = [gross == 1.0, net == 0.0, factor_exposure == 0]
weights = minimize_distance(raw, constraints)
orders = rebalance(current_weights, weights)
backtesting-evaluation

Performance Attribution

Break PnL into factor returns, stock selection, sizing, timing, and trading effects. This prevents confusing luck or hidden beta with skill.

Trading use: after every backtest or live period, identify which part of the system created or destroyed value.

pnl = portfolio_returns(weights, returns)
factor_pnl = exposures(weights) @ factor_returns
selection_pnl = pnl - factor_pnl
report = attribution(pnl, factor_pnl, selection_pnl, costs)

Stop-Loss Policy

A stop-loss policy is not just an order type. It is a governance rule for review, reduction, or exit when losses exceed expected behavior.

Trading use: define when a position is cut, resized, or escalated for review.

drawdown = position_pnl / capital_at_entry
if drawdown < hard_stop:
    target_weight = 0
elif drawdown < review_stop:
    target_weight = reduce(weight, by=0.5)
    flag_for_review(position)

Sustainable Leverage

Leverage should be set by survivability: volatility, drawdown tolerance, diversification, and business return requirements — not by broker maximums.

Trading use: cap gross exposure so the strategy can survive adverse regimes.

expected_vol = estimate_portfolio_vol(weights)
max_leverage_by_vol = target_vol / expected_vol
max_leverage_by_drawdown = loss_budget / stress_loss(weights)
leverage = min(max_leverage_by_vol, max_leverage_by_drawdown)

Factor-Neutral Alpha Portfolio

The first natural strategy from the ingestion: generate alpha scores, size them by risk, then neutralize unwanted factors before trading.

Trading use: a clean first research-to-code path for the virtual hedge fund.

alpha = compute_alpha(universe)
raw_weights = size_alpha(alpha, risk, confidence)
neutral_weights = neutralize_factors(raw_weights, exposures)
orders = apply_cost_and_liquidity_constraints(neutral_weights)
